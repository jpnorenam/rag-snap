## Quick start

### Prerequisites
Before starting, the RAG snap depends on OpenSearch and Inference snaps:

1. Install the [OpenSearch snap](https://github.com/canonical/opensearch-snap).
2. Install a [Inference snap](https://github.com/canonical/inference-snaps) of your selection.

### Installation

```bash
sudo snap rag
```
